# Spinut of the Dead
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux-lightgrey)]()

> The undead have risen... Fight through the hordes and escape Spinut to survive.

## Requirements
- Raylib
- Make

## Installation
### Clone the Repository

```bash
git clone https://github.com/JanR525/Zombie-Survival-Game/
cd Zombie-Survival-Game
```

### Building the project
#### Windows
```cmd
nsdodati
```

#### Linux
```bash
./build.sh
```

## Running the Game
```bash
nadodati
```

## Controls

| Key | Action |
|-----|--------|
| `WASD` | Movement |
| `Mouse` | Aim |
| `Left Click` | Shoot |
| `R` | Reload |
| `E` | Interact/Pick up items |
| `ESC` | Pause menu |
| `Tab` | Inventory menu |
